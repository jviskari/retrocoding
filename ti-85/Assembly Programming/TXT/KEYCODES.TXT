Keys of interest:

up    = $04 = 04
down  = $01 = 01
left  = $02 = 02
right = $03 = 03

enter = $09 = 09
2nd   = $36 = 54
exit  = $37 = 55
more  = $38 = 56

on    = no key code is returned
